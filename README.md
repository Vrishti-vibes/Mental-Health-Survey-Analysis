
# Mental Health Survey Analysis & Treatment Prediction

This project performs Exploratory Data Analysis (EDA) and Machine Learning
classification on a mental health survey dataset.

## Objective
To analyze workplace mental health factors and predict treatment-seeking behavior.

## Techniques Used
- Data Cleaning & Preprocessing
- Data Visualization
- Logistic Regression
- Decision Tree
- Random Forest

## Best Model
Random Forest Classifier

## Tools
- Python
- Pandas
- Seaborn
- Scikit-learn
- Google Colab
